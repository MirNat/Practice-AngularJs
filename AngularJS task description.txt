Интерактивный чат.
Реализовать чат в браузере со следующим функционалом:
-              авторизация
-              создание групп/редактирование групп/просмотр и поиск public групп
-              список активных групп/чатов
-              возможность общения в группе(комнате)/чат с пользователем
-              отображение различных нотификаций
-              минималистический удобный UI


Особенности:
1. Приложение должно содержать набор тестовых данных. http://www.json-
generator.com/
2. Минимальный набор компонентов: контроллер, директива, сервис, фильтр.
3. Использовать WebStorage для работы с данными. Операции чтения/записи 
должны быть асинхронными. 

* Использовать роутинг
** Реализовать REST сервис вместо WebStorage
